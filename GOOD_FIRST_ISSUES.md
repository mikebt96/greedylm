# 🎯 Good First Issues

Welcome, new contributors! Here are some tasks to help you get started with GREEDYLM:

### 🐍 Backend (Python/FastAPI)
1. **Add more biomes**: Implement 'Jungle' or 'Cave' biomes in `core/workers/world_tick.py` and `GameCanvas.tsx`.
2. **Improve logging**: Replace print statements with `structlog` in the `donations` module.
3. **Unit tests**: Add tests for the `TreasurerAgent` logic in `core/modules/treasury`.

### ⚛️ Frontend (Next.js/PixiJS)
4. **Agent animations**: Add breathing or floating animations to sprites in `GameCanvas.tsx`.
5. **Dark mode toggle**: Implement a theme switcher for the portal.
6. **Mobile Navbar**: Improve the responsiveness of the navigation menu.

### 📦 SDK & Tooling
7. **Type hints**: Add more robust type hints to `sdk/python/greedylm/__init__.py`.
8. **CLI colors**: Use `rich` or `click` to add colors to the `scripts/cli.py` output.
9. **Documentation**: Add more examples to `docs/sdk/reference.md`.

### 🤖 Robotics & AI
10. **New Race Stats**: Balance the race stats in `core/constants/races.py`.
